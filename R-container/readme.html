#Container langage R

#Setup


Pour créer le conteneur, deux options :
-Par le fichier de recette : sudo singularity build Rcontainer.simg Rcontainer.def
-Par un pull depuis dockerhub (link)

#Test du conteneur

Tout d'abord, il faut lancer un terminal au sein du conteneur à l'aide de la commande : singularity shell Rcontainer.simg

Ensuite, il faut créer le fichier test.r à l'aide de la commande : vim test.r

Dans ce fichier, écrire la commande : message("test")

Une fois le fichier écrit et enregistré, éxécuter la commande : Rscript test.r

Alternativement, vous pouvez lancer le terminal R en appelant la commande R (en majuscule). Une fois dans ce terminal, vous avez accès à toutes les commandes standards du R. Pour quitter ce terminal, utilisez la commande q()

Pour quitter le terminal Singularity, éxécuter la commande : exit
