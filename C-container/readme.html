#Container C/C++ (plus librairies Ncurses et Pthread)

#Setup


Pour créer le conteneur, deux options :
-Par le fichier de recette : sudo singularity build Ccontainer.simg Ccontainer.def
-Par un pull depuis dockerhub (link)

#Test du conteneur


Une fois le conteneur créé, il faut vérifier son fonctionnement.
Tout d'abord il faut créer un terminal dans notre conteneur à l'aide de la commande suivante : singularity shell Ccontainer.simg

Ensuite, il faut créer un fichier main.c qui contiendra le code de test. Pour cela : vim main.c
Dans ce fichier, ajouter le code suivant :
#include <stdio.h>
int main(){
printf("test\n");
return 0;
}

Ensuite, il faut créer un makefile : vim makefile
Dans ce makefile, ajouter le code suivant :
all:
gcc -o main main.c

Une fois le makefile écrit et enregistré, éxécuter la commande : make all

Ensuite, éxécuter le programme compiler en rentrant la commande : ./main

Si tout s'est bien déroulé, le terminal devrait afficher "test".

Pour quitter le terminal Singularity, éxécuter la commande : exit
